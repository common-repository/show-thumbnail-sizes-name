List of image size names and dimensions of your installation.

== Description ==

**Don't work? Open ticket, we answer in max 48h.**

You can easy view all image size names and dimensions in a settings page.

**Need to manage sizes generation?** Use our plugin [Disable Thumbnails, Threshold and Image Options](https://uskgm.it/disable-thumb-image-plugin "Disable Thumbnails, Threshold and Image Options")


== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins page directly.
2. Activate the plugin through the 'Plugins' page in WordPress.
3. Show plugin page in Tools -> Thumbnail Size.

== Frequently Asked Questions ==

= Where I can view list of thumbnails image sizes? =

You can found plugin page inside Tools -> Image Sizes.
